## How to connect to the AWS instance.

Download [RQMApp.tar.gz](https://dl.dropboxusercontent.com/u/53637793/RQMApp.tar.gz)

Unarchive using the command:
`tar -xzf RQMApp.tar.gz`

Connect using the command:
`./rqm_connect`
